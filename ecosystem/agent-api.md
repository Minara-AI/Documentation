# Agent API

{% hint style="warning" %}
This feature is still under development
{% endhint %}

Minara Agent API offers a cutting-edge financial agent service tailored for developers, traders, and market analysts. This API provides an array of tools and data for enhancing trading strategies and financial insights.

### Key Features

* **Real-Time Insights**: Access powerful and real-time data on crypto and stocks to make informed decisions.
* **AI Trading Suggestions**: Receive instant, AI-driven trading recommendations tailored to diverse trading strategies.
* **Chat-to-Execution Flow**: Benefit from seamless integration from chat to trade execution.
* **Comprehensive Services**: Includes wallet tracking, DeFi features, and alpha hunting capabilities, ready for use with agents.

### API Access

Developers can access the Minara API endpoints through two main methods:

1. **Subscription**: Available with a **Business plan**, providing comprehensive access with enhanced features. We will roll out to other plans in the future.
2. **Pay-As-You-Go**: A flexible, permissionless access option facilitated through the [x402](https://www.x402.org/), allowing usage-based payments.
