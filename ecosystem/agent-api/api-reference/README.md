# API Reference

Minara Agent API supports two ways to access endpoints:



* **API Key (Pro / Partner):** Best for production integrations. Authenticate with `Authorization: Bearer <API_KEY>` and calls consume your account credits.



* **x402 (Pay-As-You-Go):** Best for permissionless, usage-based access. Calls may require completing an x402 payment flow and attaching a payment proof (see **Authentication** / **x402 Protocol**).
